# User Funnel Analysis & Conversion Optimisation

## Project Overview

This project analyses user behaviour across an e-commerce conversion funnel.

The objective is to identify where users drop off during the purchase journey and uncover opportunities to improve conversion performance.

---

## Business Problem

Many users browse products but never complete a purchase.

The business wants to understand:

- How users move through the funnel
- Where the largest drop-offs occur
- Which product categories generate the most engagement
- Which brands attract the most customer attention

---

## Dataset

E-Commerce Behaviour Dataset

User Events:

- View
- Cart
- Purchase

Records Analysed:

174,159 user interactions

## Key Findings

### Major Drop-off

95.2% of users leave before adding products to the cart.

This suggests opportunities to improve:

- Product pages
- User experience
- Product recommendations

### Cart Conversion

29.6% of users who added products to cart completed a purchase.

Cart-stage performance is relatively strong.

## Business Recommendations

### Improve Product Page Experience

Focus on increasing View → Cart conversion.

### Personalised Recommendations

Recommend similar products based on browsing behaviour.

### Cart Retargeting

Target users who add products to cart but fail to purchase.

### Loyalty Programme

Reward high-value repeat customers.

## Business Impact

The analysis identified that the largest conversion bottleneck occurs before users add products to their cart.

Improving View-to-Cart conversion would likely generate the greatest uplift in overall revenue.
